Shapes

Canvas

CS1 Reference Link: 1.1.2

All shapes will be drawn on a canvas.
  • The point (0, 0) is the left-top corner of the canvas (not the middle!).
  • As x increases, you head right. As y increases, you head down (not up!).
  • The canvas is 400x400. So (200, 200) is the middle of the canvas, and (400, 400) is the right-bottom corner.


Shapes Objects

Rect

CS1 Reference Link: 1.1.2 or 1.3.1

To create a Rect in code, we write: 
Rect(left, top, width, height, fill='black', border=None, borderWidth=2,
     opacity=100, rotateAngle=0, dashes=False, align='left-top', visible=True)

The first four values are required, the rest are optional.

Example: 
# This creates a black rectangle with a left-top corner at (200, 200),
# a width of 20 pixels, and a height of 40 pixels.
# This makes the left 200, top 200, right 220, and bottom 240.
Rect(200, 200, 20, 40)

Oval

CS1 Reference Link: 1.3.1

To create an Oval in code, we write: 
Oval(centerX, centerY, width, height, fill='black', border=None, borderWidth=2,
     opacity=100, rotateAngle=0, dashes=False, align='center', visible=True)

The first four values are required, the rest are optional.

Example: 
# This creates a black oval with a center at (200, 200),
# a width of 60 pixels, and a height of 30 pixels.
# This makes the left 170, top 185, right 170, and bottom 215.
Oval(200, 200, 60, 30)

Circle

CS1 Reference Link: 1.3.1

To create a Circle in code, we write: 
Circle(centerX, centerY, radius, fill='black', border=None, borderWidth=2,
       opacity=100, rotateAngle=0, dashes=False, align='center', visible=True)

The first three values are required, the rest are optional.

Circle-Specific Properties:
radius

radius defines the length of half the circle.

Example: 
# This creates a black circle centered at (200, 200) with a radius of 20
# This makes the left 180, top 180, right 220, and bottom 220.
Circle(200, 200, 20)

Line (and Arrow)

CS1 Reference Link: 1.3.1 or 8.1.1

To create a Line from the point (x1, y1) to the point (x2, y2) in code, we write: 
Line(x1, y1, x2, y2, fill='black', lineWidth=2, dashes=False,
     opacity=100, visible=True, arrowStart=False, arrowEnd=False)


The first four values are required, the rest are optional.

Line-Specific Properties:
x1, y1, x2, y2, lineWidth, arrowStart, arrowEnd, dashes
  • x1, y1, x2, y2 are the x and y coordinates of the endpoints of the line.
  • lineWidth determines how thick the line is and must be bigger than 0.
  • dashes can be used to create a dashed line.
  • arrowStart can be set to True to draw an arrow head on the (x1, y1) end of the line.
  • arrowEnd can be set to True to draw an arrow head on the (x2, y2) end of the line.


Example: 
# Creates a double sided arrow with the line stretching down the center of the canvas
# from (0, 200) to (400, 200).
Line(0, 200, 400, 200, arrowStart=True, arrowEnd=True)

Label

CS1 Reference Link: 1.3.3

To create a Label with the text "value" in code, we write: 
Label(value, centerX, centerY, size=12, font='arial', bold=False,
      italic=False, fill='black', border=None, borderWidth=2, opacity=100, rotateAngle=0,
      align='center', visible=True)

The first three values are required, the rest are optional.

Label-Specific Properties:
value, font, size, bold, italic
  • value can be any type and is the text that is drawn onto the canvas.
  • font changes the font of the text with a default font of 'arial'.
  • font can either be 'arial' or 'monospace'.
  • size will change the size of the text with a default size of 12.
  • bold and italic can either be True or False with a default state of False.


Example: 
# Writes Hello World onto the canvas in arial font with the text centered at (200, 200).
Label('Hello World', 200, 200)

Regular Polygon

CS1 Reference Link: 1.3.5

To create a RegularPolygon with points number of sides in code, we write: 
RegularPolygon(centerX, centerY, radius, points, fill='black',
               border=None, borderWidth=2, opacity=100, rotateAngle=0,
               dashes=False, align='center', visible=True)

Only the first four values are required, the rest are optional.
A RegularPolygon is defined to fit exactly inside a circle of the same radius with points equi-distant around the edge of the Circle.

RegularPolygon-Specific Properties:
radius, points
  • radius is the size of the circle that the polygon fits inside.
  • radius must be a number greater than 0.
  • points is the number of points evenly spread out around the circle to create a polygon.
  • points must be an integer greater than 2.


Example: 
# Creates a triangle centered at (200, 200) with a radius of 20.
RegularPolygon(200, 200, 20, 3)

# Creates a square centered at (200, 200) with a radius of 20
RegularPolygon(200, 200, 20, 4)

Star

CS1 Reference Link: 1.3.5

To create a Star with points number of points in code, we write: 
Star(centerX, centerY, radius, points, fill=black, border=None, borderWidth=2,
     roundness=None, opacity=100, rotateAngle=0, dashes=False, align=center, visible=True)

Only the first four values are required, the rest are optional.

Star-Specific Properties:
radius, points, roundness
  • radius determines how large the star is and must be greater than 0.
  • points determines how many points are in the star and must be an integer greater than 2.
  • roundness changes how round the star is with 0 being the minimum and 100 being the maximum.


Example: 
# Creates a typical 5 pointed star centered at (200, 200).
Star(200, 200, 30, 5)

Polygon

CS1 Reference Link: 1.3.7

To create a Polygon with points (x1, y1), (x2, y2), (x3, y3), ... in code, we write: 
Polygon(x1, y1, x2, y2, x3, y3, , fill=black, border=None, borderWidth=2,
        opacity=100, rotateAngle=0, dashes=False, visible=True)

There are no required values for Polygons, they are all optional.
The polygon is drawn using straight lines in between each point, with a line between the last point and the first point as well.
Note: align is not a valid property for polygon.

Example: 
# Creates a 5-sided polygon in the top left corner of the canvas.
Polygon(25, 25, 150, 25, 200, 200, 100, 250, 25, 200)


Polygon-Specific Methods:
.addPoint(x, y)
  • addPoint takes in two numbers (x, y) and adds that point to the end of a polygon's list of points.



Example: 
# This creates a square shape with a triangle cut from it.
a = Polygon(0, 0, 0, 200, 200, 200, 200, 0)
a.addPoint(100, 100)

Arc

CS1 Reference Link: 8.1.1

To create an Arc that goes from the startAngle to the sweepAngle in code, we write: 
Arc(centerX, centerY, width, height, startAngle, sweepAngle,
            fill='black', border=None, borderWidth=2, opacity=100,
            rotateAngle=0, dashes=False, visible=True)

The first 6 values are required, the rest are optional.
An Arc is created by taking an Oval with the same centerX, centerY, width, and height and then only drawing the portion from the startAngle for sweepAngle many degrees.

Arc-Specific Properties:
startAngle, sweepAngle
  • startAngle is the angle that the cut of the oval starts at.
  • sweepAngle is the amount that is added to the startAngle to get how much of the oval the arc covers.
  • An angle of 0 degrees points straight up, and increasing the angle moves clockwise. So 90 degrees is straight right.


Example: 
# This creates an Arc that is the right-top quarter of the oval.
# The arc begins at (200, 150), and then goes 90 degrees till it hits (300, 200).
Arc(200, 200, 200, 100, 0, 90)


General Shape Properties

CS1 Reference Link: 2.3.1

General shape properties are properties which apply to all shapes. They are used to
change the shape they apply to. They can be set either when the shape is created or later on.

To set the property when creating the shape, add the following after the required properties have been set. 
propertyName = newValue

To set the property later on use: 
variableName.propertyName = newValue

Position Properties:

left, right, top, bottom, centerX, centerY
  • These properties are defined in relation to the "bounding box" of the shape, which is the smallest rectangle that completely encloses the shape.
  • centerX is the x-coordinate and centerY is the y-coordinate of the center of the bounding box of the shape.
  • Assigning a value to one of these properties causes the shape to move.

Size Properties:

width, height
  • width affects how large a shape is horizontally and height affects how large a shape is vertically.
  • width and height must both be greater than 0.

Fill:

fill
  • fill changes the color of the shape. It can be set to either a color or None.
  • The color can be a pre-defined color, a gradient, or a rgb value.
  • Fill will default to 'black' if not set otherwise.

Opacity:

opacity
  • opacity changes the transparency of the shape.
  • opacity has a default value of 100, and can go from 0 to 100.
  • 0 makes the shape invisible (completely see through).
  • 100 makes the shape completely solid (nothing can be seen behind it).

border Properties:

border, borderWidth, dashes
  • border affects the color of the border around the shape.
  • border is default set to None.
  • borderWidth affects how thick the border is. borderWidth will default be set to 2.
  • dashes makes the border into a dashed border.
  • dashes is default set to False; however it can be set either to True or to a tuple (dashWidth, gapWidth).

Align:

align
  • align changes what the position values mean when a shape is created.

Example: 
# This draws a Circle with a center at (200, 200) and a radius of 50.
Circle(200, 200, 50)

# This draws a Circle with a left-most point of (200, 200) and a radius of 50.
# Its center is at (225, 200).
Circle(200, 200, 50, align='left')
  • align can be set to: center, left, right, top, bottom, left-top, right-top, left-bottom, right-bottom
  • align must be set upon creation of the shape and cannot be changed later.
  • Polygons cannot have the align property.

RotateAngle:

rotateAngle
  • rotateAngle rotates the shape clockwise.
  • rotateAngle can be set to any number which represents the amount that the shape is being turned.
  • rotateAngle has a default value of 0 with 360 returning the shape back to its original rotation.

Visible:

visible
  • visible makes a shape visible or invisible by removing or putting the shape back on the canvas.
  • visible can be set to True or False with a default value of True.


Colors and Gradients

CS1 Reference Link: 1.2.3

Colors can be used either by using predefined colors or rgb values.

rgb Values

An rgb value is a way of representing a color by stating how much red, green, and blue appears in that color. The amount of each color is represented by a number between 0 and 255, where 0 means none of that color and 255 means all of that color.

To create an rgb value in code, we write: 
rgb(redValue, greenValue, blueValue)

Here are some example rgbs: 
# No amount of any color: black
Circle(100, 200, 50, fill=rgb(0, 0, 0))

# All of each color: white
Circle(150, 200, 50, fill=rgb(255, 255, 255))

# Only the redValue has a non-zero color: brightest red
Circle(200, 200, 50, fill=rgb(255, 0, 0))

# Only the greenValue has a non-zero color: brightest green
Circle(250, 200, 50, fill=rgb(0, 255, 0))

# Only the blueValue has a non-zero color: brightest blue
Circle(300, 200, 50, fill=rgb(0, 0, 255))

Gradients

A gradient is a smooth transition of colors. Gradients can be used to colors shapes and borders just like with single colors.

To create a gradient in code, we write: 
 gradient(color1, color2, color3, ..., start='center')

  • Gradients need at least 2 colors and can have up to as many colors as you want it to.
  • The gradient start can be used to define where the first color begins and then the gradient will proceed linearly or radially across the shape.
  • start is set to 'center' by default and can be set to all of the same values as align.

Example: 
# A gradient from the left-top to the right-bottom.
Rect(75, 75, 50, 50, fill=gradient('red', 'orange', 'yellow', start='left-top'))

# A gradient from the top to the bottom.
Rect(175, 75, 50, 50, fill=gradient('red', 'orange', 'yellow', start='top'))

# A gradient from the rightt-top to the left-bottom.
Rect(275, 75, 50, 50, fill=gradient('red', 'orange', 'yellow', start='right-top'))

# A gradient from the left to the right.
Rect(75, 175, 50, 50, fill=gradient('red', 'orange', 'yellow', start='left'))

# No start is set, so by default this is a radial gradient from out of the center.
Rect(175, 175, 50, 50, fill=gradient('red', 'orange', 'yellow'))

# A gradient from the right to the left.
Rect(275, 175, 50, 50, fill=gradient('red', 'orange', 'yellow', start='right'))

# A gradient from the left-bottom to the right-top.
Rect(75, 275, 50, 50, fill=gradient('red', 'orange', 'yellow', start='left-bottom'))

# A gradient from the bottom to the top.
Rect(175, 275, 50, 50, fill=gradient('red', 'orange', 'yellow', start='bottom'))

# A gradient from the right-bottom to the left-top.
Rect(275, 275, 50, 50, fill=gradient('red', 'orange', 'yellow', start='right-bottom'))


General Shape Methods

These methods are pre-defined methods that apply to all shapes.

To call a method in code, we write: 
variableName.method(parameters)

.toBack()

CS1 Reference Link: 4.3.2

Pushes the shape to the back behind everything else on the canvas at that moment.

Example: 
# Sends topBlue behind bottomRed.
bottomRed = Circle(200, 200, 70, fill='red')
topBlue = Circle(200, 200, 50, fill='blue')
topBlue.toBack()
topBlue.radius = 90

.toFront()

CS1 Reference Link: 4.3.2

Pulls the shape to the front of everything else on the canvas at that moment.

Example: 
# Sends bottomRed in front of topBlue.
bottomRed = Circle(200, 200, 70, fill='red')
topBlue = Circle(200, 200, 50, fill='blue')
bottomRed.toFront()
bottomRed.radius = 30

.contains(x, y)

CS1 Reference Link: 4.3.4

Takes in two numbers and returns True if the point represented by those numbers (x, y) is contained inside the shape.
Otherwise it returns False.

Example: 
# The red ring becomes gold since the point (200, 200) is inside the red ring.
# The first conditonal is True.
ring = Circle(200, 200, 100, fill=None, border='red', borderWidth=50)
if (ring.contains(200, 200) == True):
    ring.border = 'gold'
elif (ring.contains(100, 200) == True):
    ring.border = 'black'

.hits(x, y)

CS1 Reference Link: 4.3.4

Takes in two numbers and returns True if the point represented by those numbers (x, y) hits what is drawn of the shape on the canvas.
Otherwise it returns False.

Example: 
# The red ring becomes black since the inside fill is None and therefore the first hits returns False.
# The outside border is also hit making the second hits return True.
ring = Circle(200, 200, 100, fill=None, border='red', borderWidth=50)
if (ring.hits(200, 200) == True):
    ring.border = 'gold'
elif (ring.hits(100, 200) == True):
    ring.border='black'

.containsShape(shape)

CS1 Reference Link: 4.3.4

Takes in a shape and returns True if the input shape is entirely contained inside the shape the method applies to.
Otherwise it returns False.

Example: 
# blueCircle is rimmed with a green border while goldCircle has no border
# since goldCircle is not fully inside the redCircle.
redCircle = Circle(200, 200, 100, fill=None, border='red', borderWidth=50)
blueCircle = Circle(200, 200, 10, fill='blue', borderWidth=5)
goldCircle = Circle(200, 100, 20, fill='gold', borderWidth=5)

if (redCircle.containsShape(blueCircle) == True):
    blueCircle.border = 'lightGreen'
if (redCircle.containsShape(goldCircle) == True):
    goldCircle.border = 'fuchsia'

.hitsShape(shape)

CS1 Reference Link: 4.3.4

Takes in a shape and returns True if the input shape hits any filled portion of the shape that the method applies to.
Otherwise it returns False.

Example: 
# goldCircle is rimmed with a fuchsia border while the blueCircle has no border.
# goldCircle hits part of the redCircle while the blueCircle only hits
# the area not filled by anything.
redCircle = Circle(200, 200, 100, fill=None, border='red', borderWidth=50)
blueCircle = Circle(200, 200, 10, fill='blue', borderWidth=5)
goldCircle = Circle(200, 100, 20, fill='gold', borderWidth=5)

if (redCircle.hitsShape(blueCircle) == True):
    blueCircle.border = 'lightGreen'
if (redCircle.hitsShape(goldCircle) == True):
    goldCircle.border = 'fuchsia'


Groups

CS1 Reference Link: 6.1.1

A group is a collection of shapes which are treated as a single shape.

To create a Group in code, we write: 
 Group(shape1, shape2, ...)

Group Properties

CS1 Reference Link: 6.1.2

Groups have all of the general shape properties except for border, borderWidth, and align.
Properties are applied to all shapes in the group.

children Property
The children of a group are all of the shapes in the group. A single shape in the group is called a child and the group is called the parent.
groupName.children will get all shapes inside the group.

Group Specific Methods

To call a method in code, we write: 
 groupName.methodName(parameters)

.add(shape)

CS1 Reference Link: 6.2.1

Takes in a shape and adds it into the group.

Example: 
# leftTopCircle and rightBottomCircle has a gold fill while the middleCircle has no fill.
# The leftTopCircle and rightBottomCircle are added into the circleGroup group resulting in circleGroup.fill changing both circles at once.
circleGroup = Group()
leftTopCircle = Circle(100, 100, 20)
middleCircle = Circle(200, 200, 20, fill=None, border='black')
rightBottomCircle = Circle(300, 300, 20)

circleGroup.add(leftTopCircle)
circleGroup.add(rightBottomCircle)
circleGroup.fill = 'gold'

.clear()

CS1 Reference Link: 6.2.1

Removes everything from the group. This will make all of the shapes invisible.

Example: 
# All of the shapes disappear when they are cleared so nothing gains a gold fill since circleGroup is empty.
leftTopCircle = Circle(100, 100, 20)
middleCircle = Circle(200, 200, 20)
rightBottomCircle = Circle(300, 300, 20)

circleGroup = (leftTopCircle, middleCircle, rightBottomCircle)
circleGroup.clear()
circleGroup.border = 'gold'

.remove(shape)

CS1 Reference Link: 6.2.1

Takes in a variable representing a shape already in the group and removes it from the group. This will make the shape invisible.

Example: 
# middleCircle gains a gold fill but leftTopCircle and rightBottomCircle disappear.
# Originally all three circles are in the group, but since leftTopCircle and rightBottomCircle are removed
# only middleCircle is left in the group.
leftTopCircle = Circle(100, 100, 20)
middleCircle = Circle(200, 200, 20)
rightBottomCircle = Circle(300, 300, 20)

circleGroup = Group(leftTopCircle, middleCircle, rightBottomCircle)
circleGroup.remove(leftTopCircle)
circleGroup.remove(rightBottomCircle)
circleGroup.fill = 'gold'

.hitTest(x, y)

CS1 Reference Link: 10.3.1

Takes in two numbers representing the point (x, y) and returns which shape the point (x, y) hits from inside the group.
If it hits nothing from the Group it will return None.
If there are multiple shapes that it might hit, it returns the top shape.

Example: 
# The star gains a gold fill but the circle does not
# since hitTest returns the top-most shape hit in the group.
circStarGroup = Group(
     Circle(200, 200, 40, fill='blue'),
     Star(200, 200, 40, 5, fill='green')
     )
result = circStarGroup.hitTest(200, 200)
if (circStarGroup != None):
    result.fill = 'gold'


App


A predefined object which represents the overall application with some predefined properties and methods.

App Properties

Custom properties can be added to app for properties that apply to the overall application.
There are also predefined properties as well.

To set the property in code, we write: 
 app.propertyName = newValue

app.background

CS1 Reference Link: 2.3.2
Changes the background color of the canvas.
It must be set to a color of some kind.

app.group

CS1 Reference Link: 6.2.1
This is a group that automatically contains all of the shapes on the canvas. All group properties and methods apply to this group.
Shapes do not need to be explictly added to app.group.

app.stepsPerSecond

CS1 Reference Link: 6.3.1
Sets how many times event functions are called per second.
It must be set to a number.

app.paused

Sets whether the application is paused or not.
Setting it to True will pause the app and setting it to False will cause the app to keep running.


App Methods

To call a method in code, we write: 
 app.methodName(parameters)

app.stop()

CS1 Reference Link: 4.3.2

Stops all events in the app from running which is equivalent to stopping the app from running.

Example: 
# The circle expands till it got to a size of 100 then
# app.stop() is called, stopping anything from running anymore.
growingCircle = Circle(200, 200, 5)
def onStep():
    growingCircle.radius += 5
if (growingCircle.radius >= 100):
    app.stop()

app.group.hitTest(x, y)

CS1 Reference Link: 10.3.1

Takes in two numbers representing the point (x, y) and returns which shape the point (x, y) hits from on the canvas.
If it hits nothing on the canvas it will return None.
If there are multiple shapes that it might hit, it returns the top shape.

Example: 
# The top most circle that (200, 200) hits is yellow so that shape
# is returned and given a green border.
Circle(200, 200, 100, fill='red')
Circle(200, 200, 80, fill='orange')
Circle(200, 200, 50, fill='yellow'))

result = app.group.hitTest(200, 200)
if (result != None):
    result.border='green'

app.getTextInput(message)

CS1 Reference Link: 9.1.1

Takes in a string message which will be shown as the prompt the user sees in the dialog box pop up.
Returns a string of whatever the user types into the prompt.

Example: 
# The label displays the value that is returned from the getTextInput method.
response = app.getTextInput('Hello World! How are you?')
Label(response, 200, 200, size=50)


Functions


Functions will run all the code inside the function whenever the function is called.
Code is inside a function if it is indented underneath where the function is defined.

Event Functions

Event based functions will be called when certain actions or events occur.

onMousePress(mouseX, mouseY)

CS1 Reference Link: 2.2.1

onMousePress is a function that is called whenever the mouse is clicked.
(mouseX, mouseY) represents the (x, y) coordinates of where the mouse was clicked on the canvas.

Example: 
# Changes the top label to the x coordinate
# and the bottom label to the y coordinate when the mouse is pressed.
xCoord = Label('', 200, 150, size=50)
yCoord = Label('', 200, 250, size=50)

def onMousePress(mouseX, mouseY):
    xCoord.value = mouseX
    yCoord.value = mouseY

onMouseRelease(mouseX, mouseY)

CS1 Reference Link: 2.2.5

onMouseRelease is a function that is called whenever the mouse is no longer held down.
(mouseX, mouseY) represents the (x, y) coordinates of where the mouse was released.

Example: 
# The top and bottom label change to the x and y coordinate of the mouse when the mouse is released.
# If the mouse is pressed at one location, but released at a different location,
# the labels will only change to the coordinates of where the mouse has been released.
xCoord = Label('', 200, 150, size=50)
yCoord = Label('', 200, 250, size=50)

def onMouseRelease(mouseX, mouseY):
    xCoord.value = mouseX
    yCoord.value = mouseY

onMouseMove(mouseX, mouseY)

CS1 Reference Link: 3.1.1

onMouseMove is a function that is called whenever the mouse is moved (but not clicked).
(mouseX, mouseY) represents the (x, y) coordinates of where the mouse is on the canvas.

Example: 
# While the mouse is moving around but hasn't been pressed,
# the top and bottom label changes to the x and y coordinate
# respectively of the mouse at that moment.
xCoord = Label('', 200, 150, size=50)
yCoord = Label('', 200, 250, size=50)
def onMouseMove(mouseX, mouseY):
    xCoord.value = mouseX
    yCoord.value = mouseY

onMouseDrag(mouseX, mouseY)

CS1 Reference Link: 3.1.1

onMouseDrag is a function that is called while the mouse is clicked and moved.
(mouseX, mouseY) represents the (x, y) coordinates of where the mouse is on the canvas.

Example: 
# While the mouse is dragged, the top label and bottom label
# constantly changes to whatever the x and y coordinate is.
xCoord = Label('', 200, 150, size=50)
yCoord = Label('', 200, 250, size=50)
def onMouseDrag(mouseX, mouseY):
    xCoord.value = mouseX
    yCoord.value = mouseY

onKeyPress(key)

CS1 Reference Link: 4.2.1

onKeyPress is a function that is called whenever a key is pressed.
key is a string corresponding to whatever key was pressed on the keyboard.

Example: 
# When a key is pressed, the label shows what key was pressed.
keyLabel = Label('', 200, 200, size=50)
def onKeyPress(key):
    keyLabel.value = key

onKeyRelease(key)

CS1 Reference Link: 4.2.1

onKeyRelease is a function that is called whenever a key stops being held down.
key is a string corresponding to what key was released.

Example: 
# When a key is released, the label shows what key was released.
keyLabel = Label('', 200, 200, size=50)
def onKeyRelease(key):
    keyLabel.value = key

onKeyHold(keys)

CS1 Reference Link: 5.2.1

onKeyHold(keys) is a function that is called repeatedly while keys are held down.
keys is a list of all keys currently being held down.

Example: 
# While keys are being held down,
# the label will show the list of all keys being held down.
keyLabel = Label('', 200, 200, size=50)
def onKeyHold(keys):
    keyLabel.value = keys


Related Function: onKeyHolds(keys, numOfCalls):

Sound

CS1 Reference Link: 12.1.2

To add a pre-existing sound to our program, we write: 
track = Sound(url)

The url parameter is required.

A sound track can be played and paused using pause and play methods.

.pause()

  • Pauses the audio track.

.play()

  • Plays the audio track from whereever it stopped.

Special play inputs:

restart, loop
  • restart=True will cause the track to play starting from the beginning.
  • loop=True will cause the track to continue looping after it ends.

Example: 
 # Loads a sound track that can be paused/played by pressing 'p'.
# The track is looped so it never stops unless it is paused.
pausedLabel = Label('Paused...', 200, 200, size=30, bold=True)
music = Sound('https://freesound.org/data/previews/152/152574_2764107-lq.mp3')

def onKeyPress(key):
  if (key == 'p'):
      if (pausedLabel.value == 'Paused...'):
          pausedLabel.value = 'Playing!'
          music.play(loop=True)
      else:
          pausedLabel.value = 'Paused...'
          music.pause()



Glossary

variable:

CS1 Reference Link: 2.2.4
A value that is used to represent something which can be changed.

global variable:

CS1 Reference Link: 2.2.4
A variable that is declared outside any function that can be used at any point either inside or outide functions.

local variable:

CS1 Reference Link: 7.1.1
A variable that is not global and can only be used and accessed within its scope (most often, inside a function or a for loop).

parameters:

CS1 Reference Link: 2.1.1
Variables that methods and functions take in upon declaration.

arguments:

CS1 Reference Link: 2.1.1
Data or values passed into calls of the function or method.

function:

CS1 Reference Link: 2.1.1
A code body that has a name. The name can be called to repeat the code body. It can take in parameters, or data to act upon.

helper function:

CS1 Reference Link: 3.3.1
A function used within another function to help that function.

for-loop:

CS1 Reference Link: 7.2.1 or 8.3.1 or 8.3.3
A loop that is run a set number of times. Generally for loops are given a start value, an end value, and a step value.

while-loop:

CS1 Reference Link: 9.3.1
A loop that keeps running until its conditional statement becomes false.

event:

A change of state that triggers an event function call. Examples include moving the mouse, or holding a key.

conditional:

CS1 Reference Link: 3.2.1 or 3.2.3 or 4.1.1 or 5.1.1 or 5.1.2
A logical statement that can evaluate to either True or False.

type:

CS1 Reference Link: 8.2.1
A classification given to variables and data.

string:

A special type that characters within single or double quotes are classified as.

integer:

A special type that whole numbers are classified as.

float:

A special type that numbers with a decimal point are classified as.

booleans:

A special type that is either True or False.

character:

A symbol like 'a', '!', '1'.

pseudocode:

Normal everyday language to describe a coding problem (instead of syntactically correct language).
